# Smart Code Explainer

Smart Code Explainer is a simple web application that helps users understand programming code easily.

Users can paste their code and get a basic explanation based on selected level.

## Features

- Code explanation
- Beginner and medium level
- Dark mode
- Copy and download option
- History storage

## Tech Stack

- HTML
- CSS
- JavaScript

## Project Structure

index.html  
style.css  
script.js

1. Download or clone the project  
2. Open index.html in browser
